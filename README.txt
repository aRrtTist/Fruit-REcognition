**********************代码说明**********************
本工程下各文件的功能如下：
fruits：水果图片测试集
log：日志文件，用于调用Tensorboard模块查看模型变化
module：用于保存训练好的模型
venv：虚拟环境
capture.jpg：相机捕捉的图片
icon.ico：应用软件图标
init.jpg：应用软件初始化背景图片
train.py：训练模型代码
main.py：调用模型的应用软件代码
**********************软件用户手册**********************
由于时间和个人能力有限，本次工程只完成了最基本的功能要
求。用户可选择读入本地JPG图片或调用电脑摄像头模组捕捉
识别。在使用过程中，如果遇到图片粘滞现象，主要是
因为各版本的GUI实现不同，只需将软件窗口先最大化再还原
即可刷新界面。
